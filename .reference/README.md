## Key Brand Rules

- **Primary color:** `#E58A0F` (Signal Amber) — CTAs, brand moments only
- **Background:** `#FAFAF8` (warm off-white) — never pure `#FFFFFF`
- **Text:** `#23211D` (Graphite primary), `#6E695F` (Fog secondary)
- **Fonts:** Cabinet Grotesk (display), Satoshi (body) — **never** Poppins, Montserrat, Roboto
- **Logo production:** Orange mark + dark wordmark. Chrome variant is presentation only — **never in production UI**
- **Max 3 amber elements per viewport. 90% neutral. Orange is the reward.**
- **Legal entity:** STEADYWRK LLC (US/JO). No Kayan Ventures branding.
- **Address:** Building 15, King Hussein Business Park, Amman, Jordan
