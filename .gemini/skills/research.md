# Research Skill

## When to Use
Use this when the user asks you to research a topic, compare alternatives, or investigate something.

## Steps
1. **Formulate 3+ search queries** from different angles
2. **Search each** and collect sources
3. **Cross-reference** findings — flag contradictions
4. **Output** a structured `.md` file to `research/` with:
   - Executive summary
   - Source comparison table
   - Key findings with dates
   - Recommendations
5. Always include pricing, limitations, and real-world adoption data where applicable

## Output Format
Write findings to `research/<topic>.md` with clear headings, tables, and source links.
