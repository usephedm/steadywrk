## Patent Claims Analysis

### Claim 1: Core Dispatch Method
**Known from public announcement**: Event-driven AI framework for autonomous dispatch.

**Likely Elements** (based on public information about Keys Inc):
- Automated work order intake from property management platforms
- Technician matching based on trade, location, availability
- Automated outreach via SMS/call
- Acceptance tracking and dispatch confirmation

**Our Differentiation Opportunities**:
- [x] **Multi-account operation under distinct personas** — Keys Inc appears single-entity; we operate 14 simultaneous company identities
- [x] **DMG Pro-specific optimization** — Platform-specific browser automation vs generic API approach
- [x] **Real-time profitability analysis before acceptance** — NTE-based margin calculation (QANAT Principle I)
- [x] **14-company simultaneous operation architecture** — Shared brain, many faces

### Claim 2: Technician Scoring System
**Known**: "Autonomously acquiring, qualifying, onboarding, and dispatching external service providers"

**Potential Coverage**:
- Performance-based ranking
- Reliability metrics
- Geographic zone matching

**Our Approach** (QANAT Principle VI — Learn or Die):
- Continuous scoring updates per job outcome
- Multi-factor composite score (reliability, quality, speed, price, responsiveness)
- H3 geospatial zone design (see technician-recruitment-scoring.md)
- Shadow-mode A/B testing of scoring algorithms

### Claim 3: Automated Acceptance Logic
**Key Question**: Does the patent cover AI-based acceptance decisions, or rule-based only?

**Our Edge**:
- LLM-powered work order analysis
- Profitability calculation with margin thresholds (default 30%)
- Trade-specific acceptance rules
- Per-account persona-based decision making

### Claim 4: Self-Expanding Service Networks
**Known from patent description**: "Autonomously acquiring, qualifying, onboarding... external service providers without manual intervention"

**This is the key innovation**: Not static technician rosters — the system grows its own network.

**Our Implementation Plan**:
- Technician onboarding via SMS (no app required)
- Referral program with incentive structure
- H3 zone-based recruitment targeting
- Background check / COI integration
