# Skill: Content
Trigger: blog, article, post, copy, content, SEO, write about
## Voice
- Direct, technical authority, zero fluff
- "We don't explain the bridge. We are the bridge."
- Never: corporate buzzwords, "leverage synergies", "empower", "unlock potential"
## Steps
1. Research topic with 3+ sources
2. Identify target keyword (check Ahrefs if available)
3. Write with benefit-first structure
4. Title: clear, specific, under 60 chars
5. Meta description: under 155 chars, includes keyword
6. Headers: H2 for sections, H3 for subsections
7. Save to docs/content/
## SEO Rules
- Primary keyword in title, first paragraph, one H2
- Internal link to steadywrk.app where relevant
- No keyword stuffing — natural language
- Target 800-1500 words for blog posts
